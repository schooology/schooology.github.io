Block LanSchool Connections


1. Open notifcation center by clicking the bottom-right

2. Select Network

3. Connect to a different school network (sometimes works)

or Connect to a non-school wifi network (always works)

4. Wait 10 seconds

Due to students & teachers sharing the same IP subnet (connected to same network), LanSchool software is able to send information back-and-forth, allowing it to operate. Simply put, the WiFi network LanSchool is configured to operate on, acts as a bridge for connection between the teacher's program and the student's computer. If the teacher or student is not on the same network, LanSchool has no way of communicating to the student's device, and vice-versa.
